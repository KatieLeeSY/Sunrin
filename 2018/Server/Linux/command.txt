⑵ 디렉터리 관련 명령어

▼ 디렉터리의 경로 설정 기호와 의미

/
최상위 디렉터리 표시, 디렉터리 구분

..
상위 디렉터리

.
현재 디렉터리

~
사용자의 홈 디렉터리

ⓛcd(Change Directory)

▼ cd 명령어 옵션

cd .
현재 디렉터리

cd /
최상위 디렉터리로 이동

cd ..
상위 디렉터리로 이동

cd ~ OR cd
사용자의 홈 디렉터리로 이동

② pwd(Print Working Directory)

pwd
Present Working Directory. 현재 사용 중인 디렉토리.

③ ls(List Segments)

▼ ls 명령어 옵션

ls
디렉토리의 목록 확인

ls -a
숨어 있는 파일 및 디렉터리까지 화면에 출력

ls -l
파일 및 디렉터리 표시, 접근에 대한 허가사항, 링크 수, 사용자, 등록명(그룹명)과 파일의크기 및 최종적인 수정된 시간 등을 세부적으로 나열하여 사용자에게 파일과 디렉터리의 많은 정보를 제공

ls -R
현재 디렉터리 목록 및 하위에 있는 파일과 디렉터리 목록을 트리 모양으로 화면에 출력

ls t*
t로 시작하는 목록을 화면에 출력

clear
화면을 깨끗하게 지워줌!

man [명령어]
manual. 명령어를 설명해줌.
